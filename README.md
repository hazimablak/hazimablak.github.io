# 🌐 Hazım Ablak - Kişisel Portfolyo (Personal Portfolio)

Bu depo, modern web teknolojileriyle geliştirilmiş, iki dil destekli (TR/EN) kişisel portfolyo web sitemi içermektedir.

## ✨ Özellikler
- **Çift Dil Desteği:** Tek tıkla Türkçe ve İngilizce arasında geçiş.
- **Modern Arayüz:** Glassmorphism ve Cyber-Dark tema tasarımı.
- **Tam Duyarlı (Responsive):** Mobil, tablet ve masaüstü cihazlarla %100 uyumlu.
- **Performans & SEO:** Optimize edilmiş görseller ve erişilebilirlik standartları.

## 🛠️ Kullanılan Teknolojiler
- **Frontend:** HTML5, CSS3 (Modern Flex/Grid), JavaScript (Vanilla)
- **Kütüphaneler:** FontAwesome (İkonlar), Google Fonts
- **Hosting:** GitHub Pages / Vercel
